# TEST PLAN

## Scope
- Editing lighting profiles
- Lighting profiles (Work, Sleep, Relax, Welcome)
- Seasonal settings (spring, summer, autumn, winter)
- “Configure once, sync everywhere”

## Approach
The Roomlight prototype is tested primarily manually. The application is run from the command line, and the key usage paths are checked using the CLI menus and by observing the output.

## Environment
- Testing is performed on a computer
- Operating system: Windows 11
- Execution environment: terminal
- Programming language: Python
- No external services or databases
- No physical lighting hardware
