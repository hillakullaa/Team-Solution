# Lighting Settings

## Lighting Configuration

### Lighting Profiles – Base Values

| Lighting mode | Base brightness (%) | Base color temperature (K) |
|--------------|----------------------|-----------------------------|
| Work         | 80                   | 4500                        |
| Relax        | 40                   | 3000                        |
| Sleep        | 0                    | 0                           |
| Welcome      | 60                   | 3500                        |

### Seasonal Settings – Multipliers

| Season | Brightness factor | Color temperature factor |
|--------|--------------------|--------------------------|
| Spring | 1.00               | 1.00                     |
| Summer | 0.90               | 0.90                     |
| Autumn | 1.10               | 1.05                     |
| Winter | 1.25               | 1.15                     |


Effective lighting values are calculated during synchronization by applying
the active seasonal multipliers to the base lighting profile values.
